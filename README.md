# Thirst For Nitro
### Выбери машину, проедь 3 круга и побей свой рекорд.
## Описание
* В игре есть 1 круговая трасса. ❗
* Выбор из двух машин (Mistubishi Eclipse и BMW E30). 🚗
* Лучший рекорд записывается в гараже. 🏆
